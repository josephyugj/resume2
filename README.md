## 🚀 核心研究與專業領域 (Research & Expertise)

### 🤖 生成式 AI 與大語言模型 (Generative AI & LLMs)
- **參數微調與評估：** 專精於大語言模型微調技術（Low-Rank Adaptation, LoRA 秩判定機制研究） 以及 AGI 基準測試（如 ARC-AGI 2 評測）。
- **AI 代理人與自動化：** 研究 AI Agents 系統建構、Vibe Coding 框架與自動化工具（如 Claude Code, Gemini CLI, NotebookLM 應用）。
- **超級算力協作：** 運用國家高速網路與計算中心（NCHC）的 **Taiwania 3** 超級電腦與 **NVIDIA H100 GPU** 叢集，進行大規模 LLM 相關研究與產學合作。

### 📈 財經科技與量化交易 (FinTech & Quantitative Trading)
- 專注於台灣股市（特別是半導體產業與台積電 TSMC）的技術面與基本面量化分析。
- 開發基於 Python 的自動化量化交易策略腳本與績效視覺化系統。

### 👁️ 電腦視覺與智慧國土應用 (Computer Vision & Smart City)
- **資安與多媒體：** 投入 Stable Diffusion 影像生成與 Deepfake 偽造影片偵測技術。
- **非破壞性檢測：** 將 AI 影像辨識、物件偵測技術結合地底雷達（GPR）與 CCTV 巡檢，應用於污水管線缺陷自動辨識。
